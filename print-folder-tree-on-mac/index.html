Mac 印出樹狀資料夾目錄結構

11月 29, 2018 分類 macOS

最近寫文章需要將專案的檔案目錄結構用樹狀印出，方便閱讀，所以找了一些相關資料，整理一下做個筆記

homebrew 上就有個 tree 的套件可以做到這個功能

安裝方法

使用方式

安裝完成後，就可以在 terminal 直接使用 tree 這個指令啦

常用參數

中文亂碼

當你用 tree 列印資料夾目錄時，可能會發現一些中文檔名會出現亂碼的情況

加上 -N 參數，就可以印出原本的字元了

設定印出層數

加上 -L 參數，並在後面加上層數的數字

列印一層

依字母排序

加上 -v 參數可以依照字母排序

其他參數

可以透過　--help 指令查詢
